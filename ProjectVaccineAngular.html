Vaxx.ID

Web App with Angular & API

Project 5 - Opleiding .net/C#

The purpose of this excercise is to create a both an API in C# and a web app with Angular that connects to your api.

Vaxx.ID is a cyberpunk inspired vaccination center app where people can request high tech augmentation vaccination to "upgrade" their bodies. The website tries to be as user friendly by allowing the user to click on the body part for which they wish to upgrade. The users can also view a 3d map that shows the locations of all our vaccination centers.
